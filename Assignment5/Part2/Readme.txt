Instructions to run the code
Open the terminal in "Part2" directory and enter the  commannd "gcc ass5_2.c -lpthread" to compile it
 To run the code type ./a in the terminal
The program runs for finite amount of time and necessary messages are displayed.
The program can be stopped using Ctrl + C command from keyboard.

You can also use "make" command to compile it and then ./ass5_2 in the terminal to run it.although it is not advised to do it because make sometimes create
conflicts with lpthread so program got stuck
 It is better to run with the normal gcc command
